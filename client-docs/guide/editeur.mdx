Le champ **Contenu de l'article** est un éditeur de texte enrichi qui prend en charge les styles de paragraphe, les listes, les images et des blocs de mise en page avancés.

## Mise en forme de base

| Outil | Usage |
| --- | --- |
| **Normal** | Paragraphe standard |
| **Titre 2 (H2)** | Grande section — apparaît dans le **sommaire** latéral (grand écran) |
| **Titre 3 (H3)** | Sous-section — apparaît aussi dans le sommaire |
| **Citation** | Mise en exergue d'une phrase |
| **Puces / Numérotée** | Listes |
| **Gras / Italique** | Emphase classique |
| **Surligné (vert)** | Accent visuel — voir [Surlignage vert](/guide/surlignage) |
| **Lien** | URL externe (`https://…`) ou interne ; les liens externes s'ouvrent dans un nouvel onglet |

<Tip>
  Alternez paragraphes courts, titres H2 pour les grandes parties, H3 pour le détail. Un article bien structuré génère un **sommaire cliquable** automatique sur la page article (visible sur ordinateur).
</Tip>

## Insérer une image dans le contenu

<Steps>
  <Step title="Ajouter un bloc">
    Cliquez **+** dans la zone de contenu
  </Step>
  <Step title="Choisir Image">
    Sélectionnez **Image**
  </Step>
  <Step title="Compléter">
    Uploadez le fichier et renseignez le **texte alternatif**
  </Step>
</Steps>

L'image s'affiche en pleine largeur dans le corps de l'article.

## Bloc « Texte + image (côte à côte) »

Pour un texte et une illustration alignés horizontalement :

<Steps>
  <Step title="Insérer le bloc">
    Cliquez **+** → **Texte + image (côte à côte)**
  </Step>
  <Step title="Rédiger">
    Rédigez le **Texte** (paragraphes Normal ou Titre 3, listes, liens, gras, italique)
  </Step>
  <Step title="Ajouter l'image">
    Ajoutez l'**Image** (obligatoire) avec texte alternatif
  </Step>
  <Step title="Positionner">
    Choisissez la **Position de l'image** : à droite ou à gauche du texte
  </Step>
</Steps>

<Info>
  Sur mobile, le texte et l'image s'empilent automatiquement. Réservez ce bloc aux passages où l'image **accompagne** vraiment le propos (portrait, schéma, photo de situation).
</Info>
