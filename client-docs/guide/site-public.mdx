Comprendre le rendu public aide à mieux préparer le contenu dans le Studio.

## Page `/blog`

| Zone | Source dans le Studio |
| --- | --- |
| **Article vedette** (grand bloc en haut) | L'article **le plus récent** par date de publication (si absente, la date de création est utilisée) |
| **Barre de recherche** | Recherche dans les titres et descriptions |
| **Filtres par tags** | Groupes et tags pré-configurés |
| **Grille d'articles (bento)** | Tous les autres articles publiés, paginés par blocs de 6 |

## Page article `/blog/[slug]`

| Élément | Source |
| --- | --- |
| Tags | Métadonnées → Tags |
| Titre surligné | Contenu → Mot(s) à surligner |
| Date | Métadonnées → Date de publication |
| Image principale | Médias → Image principale |
| Sommaire latéral | Généré automatiquement à partir des **Titre 2** et **Titre 3** du contenu |
| Corps de l'article | Contenu de l'article |
| Galerie | Médias → Galerie d'images |
| Articles similaires | Sélection automatique selon les tags partagés et l'auteur |

<Tip>
  L'article vedette est toujours le **plus récent** par date de publication. Ajuster la date le remonte en tête, mais change aussi l'ordre dans la liste.
</Tip>
